# 🎨 Mixbox 水彩调色板 - Photoshop 插件

一个基于物理混色算法的 Photoshop 调色板插件，使用 [Mixbox](https://scrtwpns.com/mixbox/) 技术模拟真实水彩颜料的混合效果。

![License](https://img.shields.io/badge/License-CC%20BY--NC%204.0-lightgrey.svg)
![Mixbox](https://img.shields.io/badge/Powered%20by-Mixbox-blue)

## ✨ 功能特点

### 🎨 专业调色板
- **温莎牛顿 Cotman 16色** - 经典水彩颜料
- **施美尔 Horadam 16色** - 专业级艺术家颜料
- **吴竹 Gansai 16色** - 日本传统水彩
- **数字艺术家调色板** - 数字绘画常用色

### 🖌️ 真实混色体验
- **物理级混色算法** - 基于 Mixbox 技术，模拟真实颜料混合
- **可调节颜料浓度** (1-50%) - 控制颜色覆盖力
- **笔刷大小调节** (2-50px) - 适应不同绘画需求
- **多种笔刷预设** 

### 🎯 便捷操作
- **取色器** - `Alt + 左键` 取前景色，`Alt + 右键` 取背景色
- **撤销/重做** - 支持最多 50 步历史记录
- **自动保存** - 画布内容和设置自动保存

## 📦 安装方法

### 方式一：直接使用（开发模式）
1. 下载本项目到本地
2. 在 Photoshop 中打开 `文件 > 脚本 > 浏览`
3. 选择 `index.html` 文件加载插件

### 方式二：UXP 插件安装
1. 使用 Adobe UXP Developer Tool 加载插件
2. 在 Photoshop 中从 `插件` 菜单访问

## 🎮 使用指南

### 基础操作
1. **选择调色板** - 点击顶部"切换颜料"按钮选择不同品牌的颜料
2. **选择颜色** - 点击色块选择前景色
3. **调节参数**
   - 笔刷大小：控制笔触的物理尺寸
   - 颜料浓度：控制颜色的覆盖强度（浓度越高，混色越明显）
4. **开始绘制** - 在混色区域自由绘制

### 高级功能
- **取色器模式**：按住 `Alt` 键，光标变为吸管
  - `Alt + 左键` → 设置为前景色
  - `Alt + 右键` → 设置为背景色
- **笔刷切换**：点击笔刷预览图标，选择或上传笔刷
- **历史记录**：使用 `↶` 和 `↷` 按钮撤销/重做


## 🛠️ 技术栈

- **混色引擎**: [Mixbox](https://scrtwpns.com/mixbox/) - 物理级颜料混合算法
- **渲染**: WebGL + Canvas 2D 双缓冲
- **框架**: Adobe UXP (Unified Extensibility Platform)
- **存储**: LocalStorage 持久化
